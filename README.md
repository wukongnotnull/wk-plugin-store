# wk-plugin-store

Private Claude Code plugin marketplace for custom skills and tools.

## 添加市场

```bash
/plugin marketplace add wukongnotnull/wk-plugin-store
```

## 安装插件

```bash
/plugin install wk-frontend-design@wk-plugin-store
/plugin install wk-html-slides-amber-classified@wk-plugin-store
```

## 插件列表

| 插件 | 描述 |
|------|------|
| `wk-frontend-design` | Frontend design skill with patterns, design tokens, and component guidelines |
| `wk-html-slides-amber-classified` | HTML slides skill for amber-classified presentations |
